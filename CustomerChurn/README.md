Customer Churn Analysis
Overview
This project uses the IBM Telco Customer Churn dataset (Kaggle) to analyze and predict customer churn.
Dataset
~7,000 customer records
Includes demographics, services, billing, and contract details
Target: Churn (Yes/No)
Approach
Data cleaning and preprocessing
Exploratory data analysis (EDA)
Feature encoding and scaling
Model training (e.g., Logistic Regression, Random Forest)
Results
Key churn factors:
Month-to-month contracts
Short tenure
High monthly charges
How to Run
Bash
pip install -r requirements.txt
python churn_model.py
Source
Kaggle – IBM Telco Customer Churn Dataset
